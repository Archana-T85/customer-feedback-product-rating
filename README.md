# Customer Feedback & Product Rating Module

## Project Overview

A full-stack e-commerce customer feedback module where users can view products, submit ratings and reviews, and see feedback from other customers.

## Tech Stack

### Frontend
- React.js
- Vite
- JavaScript
- CSS

### Backend
- Java
- Spring Boot
- Spring Data JPA
- REST API

### Database
- MySQL

### Tools
- VS Code
- Eclipse
- Postman
- Git & GitHub

## Features

- Display product list
- View product details
- Submit product ratings
- Add customer reviews
- View existing reviews
- Calculate average product rating
- Backend validation
- REST API integration
